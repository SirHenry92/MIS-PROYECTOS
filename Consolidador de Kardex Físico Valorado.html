Consolidador de Kardex, Redacción y Foliación Automática

Sube los PDFs. El sistema identificará los Kardex normados y los archivos con nombres libres. Podrás configurar parches de forma visual para cada uno, definir el número de folio inicial y aplicar rangos de páginas físicas de forma unificada.

1. Rueda del Mouse: Zoom | Tecla Control (Ctrl) + Arrastrar: Mover vista.
2. Clic normal y arrastrar: Dibuja el cuadro rojo para tapar el dato original. ¡Parche configurado para este archivo! ✅
